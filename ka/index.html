Gogo Code

Gogo Code არის პროგრამა, რომელიც საქართველოში თინეიჯერ გოგონებს ასწავლის პროგრამირების საფუძვლებს და მხარს უჭერს მათი ტექნოლოგიებით დაინტერესებას.

რა არის Gogo Code?

Gogo Code არის პროგრამა, რომელიც საქართველოში თინეიჯერ გოგონებს ასწავლის პროგრამირების საფუძვლებს და მხარს უჭერს მათი ტექნოლოგიებით დაინტერესებას.

Gogo Code არის 10-დღიანი ბანაკი გოგონებისთვის საქართველოში. ის არის უფასო და მასში მონაწილეობის მიღება შეუძლიათ 14-16 წლის გოგონებს.

ბანაკში

მონაწილეები ისწავლიან:

  • პროგრამირების შესავალს, HTML-სა და CSS-ს
  • სამეცნიერო ექსპერიმენტების ჩატარებას
  • ვიდეო დღიურების შექმნას
  • შეიძენენ ჯგუფური მუშაობის, პრეზენტაციების გაკეთების, პრობლემის გადაჭრისა და კრიტიკული აზროვნების უნარ-ჩვევებს

რატომ პროგრამირება და ტექნოლოგიები?

საქართველოში ქალები ნაკლებად არიან ჩართულნი მეცნიერების, ტექნოლოგიების, ინჟინერიისა და მათემატიკის სპეციალობებში. გოგონები ამ საგნების გამოცდებში არ ჩამოუვარდებიან ბიჭებს, თუმცა ნაკლებ მხარდაჭერას იღებენ, არ არიან საკუთარ შესაძლებლობებში დარწმუნებულნი და, შედეგად, ინფორმაციული ტექნოლოგიების ფაკულტეტის სტუდენტების მხოლოდ 16%-ს შეადგენენ.

16%

პროგრამირება ცხოვრების ნებისმიერ ასპექტში არის მნიშვნელოვანი, იქნება ეს საზოგადოებრივი განვითარება, მედიცინა თუ ეკონომიკა.
